# TypeScript project 

## How to use

* exec `./setup.sh`
	* download dependencies.
	* required *npm*, *bower*, *tsd*.

* exec `grunt`
	* build project.
	* required *grunt.js*.
* exec `grunt clean`
	* clean up working files.
* exec `grunt test`
	* run client-side test and server-side test.
	* required *PhantomJS*.

* exec `./run.sh`
	* build & launch server.
	* required *foreman*.
* exec `npm start`
	* launch server
